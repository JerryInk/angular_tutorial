## Неделя 6. Атрибутивные директивы и кастомные Пайпы
------------------------------
## ЛЕКЦИЯ 6: Модификация поведения DOM и декларативная трансформация данных в Angular 22
Продолжительность: 2 академических часа (90 минут)
## 1. Анатомия Директив: Встроенные vs Кастомные (20 минут)
В Angular 22 все директивы разделяются на три ключевые категории: компоненты (директивы с шаблоном), структурные директивы (управляют разметкой, заменены на встроенный Control Flow @if/@for) и атрибутивные директивы.

* Атрибутивные директивы: Изменяют внешний вид или поведение уже существующего HTML-элемента, компонента или другой директивы. Они не создают новых узлов в DOM-дереве, а лишь модифицируют текущие.
* Встроенные директивы: ngClass (динамическое добавление/удаление CSS-классов) и ngStyle (инлайн-стили). С появлением сигналов и OnPush их использование часто заменяется прямой привязкой к свойствам: [class.active]="isActive()" или [style.color]="statusColor()".
* Кастомные директивы: Применяются, когда логику поведения DOM-элемента нужно инкапсулировать и повторно использовать на разных страницах проекта (например, автовыравнивание высоты текстового поля, маска ввода телефона, кастомный тултип).

## 2. Работа с DOM через ElementRef и декораторы HostListener / HostBinding (25 минут)
Для создания безопасной кастомной директивы Angular предоставляет абстракции над нативным DOM браузера:

* ElementRef: Ссылка на конкретный DOM-элемент, к которому применили директиву. Доступ к нативному узлу осуществляется через this.el.nativeElement.
* HostListener: Декоратор, позволяющий директиве «слушать» события, происходящие на элементе-хозяине (например, клик, наведение мыши, нажатие клавиши).
* HostBinding: Декоратор, связывающий свойство класса директивы со свойством или CSS-классом DOM-элемента.

Важное замечание для Angular 22: Вместо классических декораторов HostListener и HostBinding в v22 активно внедряется функциональный подход через свойство host: {} в метаданных декоратора @Directive. Это делает код чище и производительнее в Zoneless-режиме.
## 3. Пайпы (Pipes): Декларативная фильтрация и форматирование (25 минут)
Пайп (конвейер) — это инструмент для преобразования данных непосредственно внутри HTML-шаблона. Исходное значение в классе TypeScript остается неизменным, меняется только его визуальное представление для пользователя.

* Синтаксис в шаблоне: {{ rawData | pipeName : argument }}
* Встроенные пайпы: date (форматирование дат под локаль), currency (вывод валют), json (отладка объектов), lowercase/uppercase.

## Чистые (Pure) и Нечистые (Impure) пайпы:
   1. Pure Pipes (Чистые, по умолчанию): Angular вызывает метод трансформации пайпа только тогда, когда меняется ссылка на входящие данные или значение примитива. Они работают невероятно быстро, так как кэшируют результат вычислений (мемоизация).
   2. Impure Pipes (Нечистые): Вызываются при каждом цикле проверки изменений компонента, даже если данные не менялись. Написание нечистых пайпов для тяжелых вычислений (например, фильтрация больших массивов) строго запрещено, так как это мгновенно замедлит приложение.

## 4. Пайпы против методов класса в шаблонах (20 минут)
Никогда не вызывайте обычные методы класса внутри интерполяции {{ getFormattedData() }} или условий @if.
В режиме OnPush без использования Сигналов вызов метода в шаблоне может приводить к его многократному холостому перезапуску. Для форматирования строк нужно использовать исключительно Pure Pipes или Computed Signals. Пайпы кэшируют результат, а обычные методы — нет.

## ПРАКТИЧЕСКОЕ ЗАНЯТИЕ 6: Создание директивы дедлайнов и кастомного пайпа сокращения текста
Продолжительность: 2 академических часа (90 минут)
Цель работы: Разработать кастомную атрибутивную директиву для динамического изменения стилей DOM-элементов при наведении и создать чистый (Pure) пайп для безопасного форматирования строк в шаблонах.
## Шаг 1: Генерация сущностей (15 минут)
Создать директиву и пайп в рамках структуры проекта с помощью Angular CLI:
```bash

  ng g p pipes/truncate-text --standalone
```
## Шаг 2: Реализация кастомной директивы (25 минут)
Директива будет автоматически подсвечивать карточки задач или курсов, если до дедлайна осталось мало времени, а также плавно менять прозрачность элемента при наведении мыши.
Открыть highlight-deadline.directive.ts и переписать код с использованием современного синтаксиса host свойств Angular 22:
```typescript
  import { Directive, ElementRef, inject, input, OnInit } from '@angular/core';
  
  @Directive({
    selector: '[appHighlightDeadline]',
    standalone: true,
    // Современный подход v22 к обработке событий и привязке стилей хоста
    host: {
      '(mouseenter)': 'onMouseEnter()',
      '(mouseleave)': 'onMouseLeave()',
      '[style.transition]': '"all 0.3s ease"'
    }
  })
  export class HighlightDeadlineDirective implements OnInit {
    private el = inject(ElementRef); // Современный функциональный инжектор
  
    // Входной сигнальный параметр: количество оставшихся дней
    daysLeft = input.required<number>({ alias: 'appHighlightDeadline' });
  
    ngOnInit(): void {
      const domElement = this.el.nativeElement as HTMLElement;
      
      // Логика модификации DOM на основе входных данных
      if (this.daysLeft() <= 2) {
        domElement.style.borderLeft = '5px solid #dc3545'; // Красный маркер (срочно)
        domElement.style.backgroundColor = '#fff5f5';
      } else if (this.daysLeft() <= 7) {
        domElement.style.borderLeft = '5px solid #ffc107'; // Желтый маркер (внимание)
      } else {
        domElement.style.borderLeft = '5px solid #28a745'; // Зеленый маркер (нормально)
      }
    }
  
    // Метод, вызываемый при наведении мыши (описан в host)
    onMouseEnter(): void {
      this.el.nativeElement.style.transform = 'scale(1.02)';
      this.el.nativeElement.style.boxShadow = '0 4px 15px rgba(0,0,0,0.1)';
    }
  
    // Метод, вызываемый когда мышь уходит с элемента
    onMouseLeave(): void {
      this.el.nativeElement.style.transform = 'scale(1)';
      this.el.nativeElement.style.boxShadow = 'none';
    }
  }
```
## Шаг 3: Реализация кастомного чистый пайпа (20 минут)
Пайп будет принимать длинное описание курса или задачи и безопасно обрезать его до указанного лимита символов, добавляя троеточие в конец.
Открыть truncate-text.pipe.ts:
```typescript
  import { Pipe, PipeTransform } from '@angular/core';
  
  @Pipe({
    name: 'truncateText',
    pure: true, // Фиксируем, что пайп чистый (включает кэширование)
    standalone: true
  })
  export class TruncateTextPipe implements PipeTransform {
    // Первый аргумент — входящая строка, второй — опциональный лимит длины
    transform(value: string, limit: number = 40): string {
      if (!value) return '';
      if (value.length <= limit) return value;
      
      // Безопасное обрезание строки
      return value.substring(0, limit) + '...';
    }
  }
```
## Шаг 4: Подключение и интеграция в UI (20 минут)
Интегрировать созданные инструменты в карточку курса/задачи (CourseCardComponent или TaskCardComponent), созданную на прошлой неделе.

   1. Открыть course-card.component.ts. Импортировать директиву и пайп прямо в массив imports компонента:
```typescript
  import { HighlightDeadlineDirective } from '../../directives/highlight-deadline.directive';
  import { TruncateTextPipe } from '../../pipes/truncate-text.pipe';

  @Component({
    // ... предыдущие настройки
    imports: [HighlightDeadlineDirective, TruncateTextPipe], // Подключаем кастомные утилиты
  })
  export class CourseCardComponent {
    // Симулируем поле дней до окончания для теста директивы
    daysToStart = signal<number>(1); 
    description = signal<string>('Длинное инженерное описание лабораторного курса по веб-технологиям и развертыванию SPA на фреймворке Angular 22.');
  }
```
   2. Открыть course-card.component.html и применить директиву как атрибут, а пайп через символ черты |:
```html
  <!-- Применяем кастомную директиву, передавая значение сигнала в квадратных скобках -->
  <div class="card-item" [appHighlightDeadline]="daysToStart()">
  
    <h4>{{ course().title }}</h4>
  
    <!-- Применяем кастомный чистый пайп с аргументом ограничения в 30 символов -->
    <p class="desc">{{ description() | truncateText : 30 }}</p>
  
    <p>Осталось дней: {{ daysToStart() }}</p>
  </div>
```
